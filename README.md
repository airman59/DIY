# DIY

Trying to build a SPA-application using MongoDB, Express.js, Node.js and
ES5/ES6. 
A live example can be found at [kingsislet.com](http://www.kingsislet.com).

The repository is for my own version-control.

## Requirements

* node (v6.7.0)
* npm (v3.10.3)
* mongodb (v3.2.7)
